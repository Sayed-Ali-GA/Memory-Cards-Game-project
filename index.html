Memory Cards Game

How to Play Memory Cards

1. Flip Two Cards:
Click on two cards to flip them over.

2. Match Pairs:
If the cards show the same symbol, you score a point!

3. Win the Game:
Match all pairs before the timer runs out to win the game.

4. Timer:
You have 20 seconds to match all pairs.
The timer will start automatically when you click "Start Game".

Good luck and have fun!
